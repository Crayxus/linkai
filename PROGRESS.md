## 已完成功能

### 核心功能
- 三科课程侧边栏（语文/数学/英语，人教版2026春）
- AI 讲解（豆包流式输出）
- AI 问答（聊天模式）
- 闯关练习（8道题，3颗心，答错扣心）
- 模拟考试（期中/期末，AI出完整试卷）
- TTS 朗读（edge-tts，中文云溪/英文 Guy）
- XP 积分 + 连续学习天数
- 错题本（按科目分类）
- 每日学习计划（新课推荐 + 到期复习）
- 艾宾浩斯遗忘曲线（1/3/7天复习）

### 本次新增（2026-03-08）
#### 1. 隐藏分数 & 考试预测系统
- 数据库新增 `kp_log` 表 — 知识点级别答题记录
- 遗忘曲线加权公式（14天半衰期）
- 三科各自计算预测分（0%掌握→50分，100%→100分）
- API: `GET /api/exam_readiness` — 返回预测分+薄弱知识点排名
- API: `POST /api/kp_log` — 记录每题知识点对错

#### 2. 薄弱突破训练
- API: `POST /api/targeted_quiz` — 针对弱点专项出题
- 全局入口：顶部"📊 成绩"按钮 → 每科"⚡ 薄弱突破训练"
- 课程入口：每节课菜单新增"薄弱突破"卡片
- 答题时自动记录知识点掌握度

#### 3. 成绩仪表盘（pg-readiness）
- 三科预测分大卡片（红/橙/绿色警示）
- "距第一还差X分"提示（目标95分）
- 每科薄弱知识点进度条排名
- 首页也注入三科预测分快览

#### 4. Bug 修复
- quiz API 格式异常 → 改用 `response_format={"type":"json_object"}` 强制 JSON
- 简化 prompt，去掉复杂的格式示例
- TTS 朗读《》书名号 → 前后端均过滤书名号
- TTS 按钮逻辑修复（点击不再空停）
- 朗读题目时同时读选项内容
- 观察物体等视觉题选项为空 → prompt 禁止图形题 + 服务端过滤空选项
